== Description ==
**S2 Donation using Stripe** allows you to receive donation using stripe. User can donate amount using cards(VISA, Master etc.).

Check some of its features:

1. See report(live data, test data) of donation i.e. how many user made donation.
2. Donation will be made using Stripe checkout form so need not worry about securities, Stripe will handle it.
3. Minimalist design.
4. Embed a donation form on any page using shortcode.
5. Recurring donation(daily, weekly, monthly, annually etc.).
6. Fixed donation amount.
7. Enable mail sending to user who made donation.


== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate "S2 Donation using Stripe" from Plugins page.


== Configuration ==
"S2 Donation using Stripe" will add a new submenu page called "Donation" in "S2 Plugins" menu item.
There you will find all "S2 plugins" with quick access to plugin pages.
